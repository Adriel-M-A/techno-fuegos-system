# Bienvenido a Techno Fuegos System

Este programa fue creado especialmente para **Techno Fuegos** (Pilar, Buenos Aires) con el propósito de **agilizar, simplificar y unificar** la creación de presupuestos comerciales para todos nuestros productos de herrería y carpintería artesanal.

## ¿Qué problema resolvemos?
Antes, para calcular el precio de un fono, portón o estructura a medida, se dependía del encargado de producción del taller para estimar los materiales y horas de trabajo. Esto provocaba:
- **Demoras para vender:** El cliente debía esperar días para recibir su cotización.
- **Precios diferentes:** A veces se cobraban precios distintos por trabajos muy similares debido a la falta de un método de cálculo unificado.

## ¿Cómo te ayuda esta aplicación?
Esta aplicación funciona directamente en tu computadora **sin necesidad de conexión a Internet**. Permite que cualquier vendedor o administrativo pueda **generar un presupuesto exacto en menos de 5 minutos**.

Al tener los precios de los materiales base (kilos de chapa, metros de perfiles, litros de pintura, horas de trabajo) estandarizados en el sistema, la aplicación realiza la suma de costos automáticamente y al instante, garantizando presupuestos precisos, justos y profesionales.
